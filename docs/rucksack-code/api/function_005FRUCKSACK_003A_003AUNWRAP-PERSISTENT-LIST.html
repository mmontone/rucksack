Function: UNWRAP-PERSISTENT-LIST

Documentation

Converts a persistent list to a 'normal' Lisp list.

Source

(defun unwrap-persistent-list (list)
  "Converts a persistent list to a 'normal' Lisp list."
  (loop until (p-endp list)
        collect (p-car list)
        do (setq list (p-cdr list))))
